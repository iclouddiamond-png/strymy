# Инструкция по настройке Telegram Bot и Админ Панели

## Шаг 1: Создание Telegram Bot

1. Откройте Telegram и найдите бота **@BotFather**
2. Напишите `/newbot`
3. Следуйте инструкциям:
   - Введите имя бота (например: "Top Modeli UA Bot")
   - Введите username бота (должен заканчиваться на "_bot", например: "top_modeli_ua_bot")
4. @BotFather отправит вам **Bot Token** - скопируйте его

## Шаг 2: Запуск сервера

В отдельном терминале выполните:

```bash
# На macOS/Linux:
cd /Users/iclouddiamond888/strymy
node server.cjs
```

## Шаг 3: Процесс регистрации пользователя

1. Пользователь вводит только номер телефона
2. Система генерирует код и deep link на бота
3. Пользователь нажимает кнопку "Открыть бота"
4. Бот отправляет код подтверждения в личный чат
5. Пользователь вводит код на сайте
6. После успешной регистрации появляется сообщение "Регистрация успешна!"

## Шаг 4: Админ Панель

**Доступ:** http://localhost:5173/admin

**Пароль:** `admin`

### Функции Админ Панели:

✅ **Просмотр всех пользователей**
- Номер телефона
- Код регистрации
- Статус верификации
- Дата регистрации

✅ **Контакт с пользователят**
- Выбрать пользователя из списка
- Отправить сообщение в Telegram
- Сообщение придет пользователю в личный чат с ботом

✅ **Управление данными**
- Удалить пользователя из базы
- Все данные хранятся в `users.json`

## Файлы данных

- **users.json** - база данных пользователей (номер телефона, код, статус)
- **chat_ids.json** - соответствие username → chat_id для отправки сообщений в Telegram

## Запуск приложения

В одном терминале запустите backend:
```bash
node server.cjs
```

В другом терминале запустите frontend:
```bash
npm run dev
```

Откройте http://localhost:5173/ в браузере.
